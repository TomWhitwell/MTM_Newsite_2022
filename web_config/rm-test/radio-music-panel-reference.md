# Radio Music Mk2 Panel Reference

## Top section
- Four main LEDs report startup status, bank display, VU/progress display, and mode feedback.
- `STATION` knob selects station/file.
- Hold `STATION` push-switch to browse/select banks.

## Middle section
- `START` knob controls start position (Start mode) or pitch/speed (Pitch mode).
- Press `START` push-switch to toggle Start/Pitch mode.
- `RESET` button retriggers playback and supports long-hold bank stepping.

## Bottom jack row
- `STATION` jack: station CV input.
- `START` jack: start CV input or pitch CV input (depends on mode).
- `RESET` jack: reset trigger input when `pulseMode=0`, pulse output when `pulseMode!=0`.
- `OUT` jack: audio output.

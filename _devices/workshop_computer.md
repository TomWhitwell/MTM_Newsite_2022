pars:
- "Workshop Computer is a simple but capable music computer, inspired by the early music studios like EMS in London and Bell Labs where computers were used to control analog oscillators and filters, compose algorithmic music and generate waveforms."
- "Because it's no longer 1968, Workshop Computer is powerful enough to do audio effects, sequencing and can connect with other devices over USB."
- "Workshop Computer loads programs stored on tiny custom-made program cards slightly larger than Micro SD cards. It's like a Game Boy for your Eurorack Modular Synth."
- "The Computer comes with four program cards: 
<ul>
<li> Reverb+ is a great-sounding old-school stereo reverb with some useful utilities including sequencing and MIDI.  
<li> Simple MIDI turns the Computer into a simple USB midi interface to send and receive MIDI. 
<li> Turing Machine is a random looping sequencer with two channels and multiple scales.
<li> It also comes with a blank program card which you can write with a downloaded file.

</ul>"
- Learn more about these cards and the whole <a href=/Computer_Program_Cards/>Workshop Computer system here</a>. 
- There are already dozens of program cards available to download, developed by a vibrant community of developers. They range from useful utilities to wild experiments. You can <a href=https://tomwhitwell.github.io/Workshop_Computer/index.html>find program cards to download here</a>.

- "Workshop Computer was originally part of the <a href=https://www.musicthing.co.uk/workshopsystem/>Music Thing Workshop System</a>, which includes a powerful analog toolkit with the Computer. The standalone Eurorack version is electronically identical and 100% compatible with the Workshop System version."

- "This video shows how to <a href=https://www.youtube.com/shorts/fYTb38B0Wsc>load your own program cards</a>." 
- "Workshop Computer has three pairs of inputs and outputs: for Audio/CV, for precision control voltages for pitch, and for pulses." 
- "Workshop Computer is a fairly straightforward DIY build. All the tiny surface-mount electronic components are pre-soldered, you only need to attach pots, switches, sockets and LEDs. If you have any experience this should be very straightforward. If it's the first thing you've ever soldered it might be worth practicing on something simpler (like a <a href=https://www.thonk.co.uk/shop/mtm-mikrophonie-bundle/>Mikrophonie</a> or <a href=https://www.thonk.co.uk/shop/mini-drive/>Mini Drive</a> kit) beforehand."

- Within the development community, people have created program cards using use C++, Arduino, Rust, Python and Lua. Find out more at <a href=https://github.com/TomWhitwell/Workshop_Computer>the developer site</a> and in the <a href=https://www.musicthing.co.uk/discord/>Workshop System Discord</a>.
